A simple and cute discord bot.

### Setup

```bash
$ git clone https://github.com/sqdorte/saturn.git
$ cd saturn/bot/
$ export TOKEN="YOUR_TOKEN"
$ export LOG="LOG_CHANNEL_ID"
$ python bot.py
```
